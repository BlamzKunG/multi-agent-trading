# Multi-Agent Trading Orchestrator

ระบบบอทเทรดทองคำ (XAUUSD) และสินทรัพย์ดิจิทัลแบบทำงานตามรอบเวลา (Stateless Loop) ขับเคลื่อนด้วยระบบสถาปัตยกรรมตัวแทนหลายตัว (Multi-Agent Architecture) ประมวลผลผ่านโมเดลภาษาขนาดใหญ่ (LLMs) และเชื่อมต่อผ่านระบบ MetaTrader 5

---

## ส่วนประกอบของระบบ (System Architecture)

ระบบถูกออกแบบมาแยกส่วนเพื่อให้ง่ายต่อการขยายและทดสอบ ประกอบด้วยไฟล์หลักดังนี้:

1. **bot_orchestrator.py**: ตัวควบคุมลูปการทำงานหลักในโหมดจำลอง (Simulation Mode)
2. **bot_orchestrator_mt5.py**: ตัวควบคุมลูปการทำงานหลักร่วมกับแพลตฟอร์ม MetaTrader 5 (Live Mode)
3. **gui_dashboard.py**: ระบบหน้าจอควบคุมหลักของแอปพลิเคชัน (Desktop Dashboard GUI) พัฒนาด้วย Tkinter สำหรับกำหนดค่าความเสี่ยง คีย์การใช้งาน และเปิด-ปิดระบบทำงานอัตโนมัติ
4. **trading_agents.py**: คลาสการทำงานร่วมกับ AI Agent ทั้งในส่วนการวิเคราะห์ทางเทคนิค (Analyst Agent) และการบริหารพอร์ตโฟลิโอและความเสี่ยง (Manager Agent)
5. **mt5_integration.py**: โมดูลสำหรับเชื่อมต่อ สั่งการเปิด-ปิดออเดอร์ ตลอดจนการทำระบบแก้ไขการตั้ง Stop Loss และ Take Profit ให้สอดคล้องกับ Stops Level ของโบรกเกอร์ (Self-Healing Logic)
6. **exchange_sim.py**: ระบบจำลองสภาวะการซื้อขายในตลาดจำลอง สำหรับทดสอบตรรกะและการคำนวณ Margin
7. **data_feed.py**: ตัวดึงข้อมูลราคาสินทรัพย์จริงผ่าน API สาธารณะ

---

## คุณสมบัติหลัก (Core Features)

- **การวิเคราะห์แบบหลายกรอบเวลา (Multi-Timeframe Analysis):** ดึงข้อมูลแท่งเทียนในกรอบเวลา 5 นาที (5m), 15 นาที (15m) และ 1 ชั่วโมง (1h) เพื่อให้ตัวแทนวิเคราะห์ตัดสินใจทิศทางร่วมกันอย่างมีประสิทธิภาพ
- **ระบบป้องกันการทำงานผิดพลาดของ Stop Loss (Self-Healing Trailing Stop):** ตรวจสอบและแก้ไขค่า Stop Loss ของออเดอร์ให้เหมาะสมกับราคา Bid/Ask ล่าสุดและ Stops Level ของโบรกเกอร์โดยอัตโนมัติก่อนส่งคำสั่งปรับปรุงไปยัง MT5 เพื่อแก้ปัญหาคำสั่งถูกปฏิเสธจากโบรกเกอร์ (Invalid Stops)
- **ระบบทำงานอัตโนมัติตามรอบเวลา (Background Scheduler):** ระบบสามารถตั้งค่ารอบเวลาการทำงานย่อย (เช่น รันทุก x นาที) ทำงานในแบบ Background Thread ช่วยให้หน้าจอโปรแกรมตอบสนองได้อย่างราบรื่น
- **การแจ้งเตือนเหตุการณ์ผ่าน Discord Webhook:** ส่งข้อความสรุปผลวิเคราะห์ คำสั่งที่ส่งเข้าตลาด ตลอดจนรายละเอียดการปรับปรุงความปลอดภัยของพอร์ตผ่านกลุ่มหรือห้อง Discord ที่ระบุไว้แบบทันที

## การเตรียมการติดตั้ง (Installation)

### 1. โคลนคลังเก็บรหัสต้นฉบับ (Repository)
```bash
git clone https://github.com/BlamzKunG/multi-agent-trading.git
cd multi-agent-trading
```

### 2. ติดตั้งไลบรารีที่จำเป็น
```bash
pip install pandas requests
```

หากต้องการใช้งานร่วมกับพอร์ตจริงผ่าน MetaTrader 5 (สามารถใช้งานได้บนระบบปฏิบัติการ Windows เท่านั้น):
```bash
pip install MetaTrader5
```

## การกำหนดค่าและการใช้งาน (Configuration and Usage)

### การตั้งค่าคีย์การเข้าถึง API
กรุณากำหนดคีย์เข้าใช้งาน MaxPlus API ในตัวแปรสภาพแวดล้อม (Environment Variables) ของระบบปฏิบัติการก่อนเริ่มต้นรันบอท:

- **Windows (Command Prompt):**
  ```cmd
  set MAXPLUS_API_KEY="คีย์ของคุณ"
  ```
- **macOS / Linux:**
  ```bash
  export MAXPLUS_API_KEY="คีย์ของคุณ"
  ```

### การเปิดใช้งานระบบควบคุมผ่านหน้าต่างแสดงผล (Desktop GUI)
รันไฟล์แสดงผลหลักเพื่อควบคุม กำหนดสิทธิ์บัญชี และตั้งเวลาทำงานอัตโนมัติ:
```bash
python gui_dashboard.py
```
เมื่อหน้าต่างโปรแกรมเปิดขึ้นมา คุณสามารถกำหนดเงื่อนไขดังต่อไปนี้ได้:
- สลับโหมดการทำงาน (Simulation / MT5 Live)
- กรอกรหัสการล็อกอิน MT5, รหัสผ่าน และโบรกเกอร์เซิร์ฟเวอร์
- กำหนดสัดส่วนจำกัดล็อตสูงสุด (Max Allowed Lot size) และรอบเวลาเป็นนาที (Interval Cycle)
- เปิดระบบความคุมอัตโนมัติ (Auto-Pilot)
- สั่งปิดออเดอร์ หรือกดสั่งการวิเคราะห์ทันทีแบบ Manual

### การทดสอบระบบซื้อขายจำลองผ่าน CLI
สำหรับทดสอบระบบในสภาพแวดล้อมจำลอง (Sandbox):
```bash
python bot_orchestrator.py
```
